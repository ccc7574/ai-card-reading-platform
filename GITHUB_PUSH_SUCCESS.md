## 🚀 **部署建议**

### **1. Vercel部署 (推荐)**
```bash
# 安装Vercel CLI
npm i -g vercel

# 连接GitHub仓库并部署
vercel --prod
```

### **2. 环境变量设置**
在Vercel中添加以下环境变量：
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
OPENAI_API_KEY=your_openai_api_key
GEMINI_API_KEY=your_gemini_api_key
```

### **3. 自定义域名**
- 在Vercel中设置自定义域名
- 配置DNS记录指向Vercel
- 启用HTTPS和CDN加速
